# DESARROLLOPROYECTO

En el proyecto de desarrollo debiamos desarrollar una aplicacion en Java que funciona como un "Sistema de monitoreo", la funcion principal de la aplicacion es realizar el monitoreo de datos de un dispositivo Arduino que se conectaba mediante USB a la computadora en cualquiera de sus puertos, lo que debia hacer la aplicacion es recibir los datos de el arduino y graficarlos en tiempo real, al recibir los datos estos debian ser encriptados y enviados a un servidor con una conexion TCP/IP, el servidor los almacenaba en una base de datos, en otra parte de la aplicacion llamada registro historico, se podian hacer consultas de las graficas por dia y hora, estas mostraban todos los datos que se leian en las horas especificadas.

Considero que este proyecto nos fue util para aprender a realizar proyectos que tengan condiciones muy especificas tal y como lo es el Arduino y aprender a tener un poco mas de capacidad para pensar y resolver problemas con dispositivos externos al programa.
